# Cinema Buffet Sales Application 🎬🍿

## 📌 Project Description
This project is a desktop application developed to manage sales operations in a cinema buffet system.

## 🧠 Technologies Used
- C#  
- Windows Forms  
- .NET Framework  

## 📊 Features
- Product sales management  
- Price calculation  
- User-friendly interface  
- Basic automation system  

## 🎯 Goal
To develop a simple automation system for managing buffet sales operations.

## 🚀 Status
Completed as a desktop application project using C#.
